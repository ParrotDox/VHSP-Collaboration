Предварительная подготовка:

1. Попросить, чтобы владелец репозитория добавил тебя в участники (по нику в Git'е)

2. Клонировать репозиторий себе на компьютер:
Для указания конкретной папки для создания репозитория используется команда cd в терминале
cd "C:\My Path\"
git clone https://github.com/ParrotDox/VHSP-collaboration.git




Кооперативная работа:

1. Добавить файл(ы) (если они существуют на репозитории, они будут изменены [важно для дальнейших действий]):
git add . - добавить все файлы
git add example.file - Добавить файл
git add "My folder" - Добавить файлы из папки
git add "My folder\example.txt" - Добавить файл из папки

2. Закоммитить свои изменения:
